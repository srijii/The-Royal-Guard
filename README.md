## About The Game

**In Search of the Ring** is a 2D story-action game where you play as a royal guard protecting the kingdom during a night invasion.

Fight enemies, explore dangerous worlds, complete quests, collect keys, and survive long enough to recover the stolen royal ring.

> The in-game title screen also uses the alternate title: **Royal Guard**

# Features

- Story-driven gameplay progression
- Dialogue and quest systems
- Multiple worlds and scene transitions
- Combat against skeletons, wizards, and boss enemies
- Health, energy, and potion systems
- Save and continue support
- Rebindable controls
- Optional mobile controls
- Final boss battle and progression system
- Built entirely using Godot 4.5 and GDScript

---

# Gameplay Progression

## 1. Palace Introduction

- Meet the princess
- Learn the kingdom's situation
- Find important quest items
- Prepare for the incoming attack

## 2. Night Invasion

- Defend the palace from skeleton thieves
- Survive enemy encounters
- Protect the royal ring

## 3. Outer World

- Explore dangerous areas
- Defeat enemies and collect keys
- Unlock progression gates

## 4. Final Boss

- Reach the final arena
- Defeat the boss
- Recover the princess's ring

# Controls

| Action | Key |
|---|---|
| Move | WASD / Arrow Keys |
| Attack | Ctrl |
| Sprint | Shift |
| Zoom Out Map | Hold M |
| Lamp Toggle | L |
| Interact | E |
| Pause | Esc |
| Show Controls | H |

---

## Potions

| Item | Key |
|---|---|
| Regeneration Potion | Space |
| Strength Potion | J |
| Energy Drink | K |

Controls can be changed from the in-game settings menu.

# Installation & Running

## Requirements

- Godot Engine **4.5**
- Forward Plus renderer recommended

---

## Clone The Repository

```bash
git clone https://github.com/YOUR_USERNAME/in-search-of-the-ring.git
```

---

## Open In Godot

1. Open Godot 4.5
2. Import the project folder
3. Open `project.godot`
4. Run the project

Main scene:

```text
scene/main_menu.tscn
```

---

# Export Support

Current export configuration includes:

- Windows Desktop

Export presets are stored in:

```text
export_presets.cfg
```

---

# Save Files

The game stores runtime files inside Godot's `user://` directory.

Files include:

```text
savegame.json
keybinds.cfg
options.cfg
```

# Project Structure

```text
scene/      -> Game scenes
script/     -> Gameplay and UI scripts
art/        -> Art assets
sounds/     -> Audio assets
tileset/    -> Tilemaps and tiles
fonts/      -> Fonts
imgg/       -> UI and miscellaneous images
```
